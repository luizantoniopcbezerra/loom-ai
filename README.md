# Loom

AI-powered project management and design tool.

## Tech Stack

- **Vue 3** — UI framework
- **Vite** — Build tool and dev server

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

| Command           | Description              |
| ----------------- | ------------------------ |
| `npm run dev`     | Start development server |
| `npm run build`   | Build for production     |
| `npm run preview` | Preview production build |
