# Loom

AI-powered project management and design tool.

## Sobre

Loom é uma ferramenta de gerenciamento de projetos e design com inteligência artificial, desenvolvida com Vue 3 no frontend e Spring Boot no backend.

## Tech Stack

- **Vue 3** — UI framework
- **Vite** — Build tool and dev server
- **Spring Boot** — Backend API
- **SQLite** — Banco de dados

## Pré-requisitos

- Node.js (versão 18+)
- Java 17+
- Maven
- SQLite

## Passo a passo

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/loom.git
cd loom
```

2. Instale as dependências do frontend:

```bash
npm install
```

3. Inicie o servidor de desenvolvimento do frontend:

```bash
npm run dev
```

4. Em outro terminal, acesse a pasta `backend` e inicie o servidor Spring Boot:

```bash
cd backend
mvn spring-boot:run
```

5. Acesse no navegador:
   - Frontend: http://localhost:5173
   - API: http://localhost:8080
